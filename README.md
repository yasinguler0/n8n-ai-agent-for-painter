About the Project

This is an autonomous agent prototype built with n8n and Gemini, designed for automated stock and cost management in painting operations.

Important Note: The V1 architecture relies entirely on AI-driven decision-making. Due to the inherent reliability issues and high error rates of agentic architectures in tool-calling processes, this version serves as a technical demo only. Future iterations will adopt deterministic workflows to ensure production-grade reliability.

Tech Stack:

Automation: n8n

LLM: Google Gemini 2.5 Flash Lite

DB: Google Sheets API

UI: Telegram Bot API
